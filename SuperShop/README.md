# Super Shop

Point of Sales and Inventory Management for SuperShop. Done as a project for Java course of CSE, SEU. 

## LOG IN 

4 kind personels can login in this Software. 
  -  Admin
  -  Manager 
  -  Stock Keeper
  -  SalesMen
